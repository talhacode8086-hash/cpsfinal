Technical Note

This tool measures the delta between the hardware event timestamp (`event.timeStamp`) and the execution of the click handler in the browser. It highlights "Input Lag" caused by CPU bottlenecks or browser thread congestion. In a clean environment, this should be between 1.0ms and 4.0ms.
